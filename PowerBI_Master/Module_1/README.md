# 📊 Sales Overview Report | Power BI Module 1

## 📌 Project Overview

This project is my first hands-on Power BI report created as part of my Power BI learning journey.

The goal of this project was to understand the Power BI workflow by importing data, creating basic visualizations, and building a simple sales report.

This project covers the fundamentals of Power BI including data import, report creation, and basic dashboard design.

---

# 🎯 Objectives

- Understand Power BI Desktop
- Import data from Excel
- Learn the Power BI interface
- Create basic visualizations
- Build the first Power BI report
- Understand the Power BI workflow

# 📊 Dashboard Features

The report contains the following visualizations:

- 📌 Total Sales Card
- 📌 Products Card
- 📌 Category Card
- 📌 Sales by Category (Bar Chart)
- 📌 Sales by Category (Pie Chart)
- 📌 Product Sales Table

---

# 📈 Business Insights

- Total Sales: ₹182,000
- Total Products: 10
- Total Categories: 2
- Furniture generated higher sales than Electronics.
- Product-wise sales are displayed in a tabular format.
- Category-wise sales comparison is visualized using Bar and Pie Charts.
