# 🛒 Retail Sales Analysis Using Python

## 📌 Project Overview

This project demonstrates an end-to-end **Retail Sales Analysis** using Python. The primary objective is to analyze retail transaction data, uncover business insights, and support data-driven decision-making through data cleaning, feature engineering, exploratory data analysis (EDA), visualization, and KPI analysis.

---

# 🎯 Project Objectives

- Clean and preprocess retail sales data
- Perform feature engineering
- Calculate business KPIs
- Conduct Exploratory Data Analysis (EDA)
- Visualize sales trends and business performance
- Generate business insights
- Provide business recommendations

---
